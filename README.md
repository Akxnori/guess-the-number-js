# 🎯 Jogo do Número Secreto 2.0

Uma aplicação web interativa onde o objetivo é simples: adivinhar o número secreto gerado aleatoriamente.

> Projeto focado em lógica de programação, manipulação de DOM e experiência interativa no navegador.

---

## 🚀 Acesse o projeto

🔗 https://jogo-puce-iota-57.vercel.app

---

## 🧠 Como funciona

- O sistema gera um número aleatório entre 1 e 10
- O jogador tenta adivinhar o número
- O sistema dá dicas em tempo real:
  - "O número é maior"
  - "O número é menor"
- O jogo contabiliza o número de tentativas até o acerto
- Após acertar, é possível reiniciar o jogo

---

## ⚙️ Funcionalidades

- Geração de número aleatório sem repetição imediata
- Feedback dinâmico para o usuário
- Controle de tentativas
- Reinício do jogo sem recarregar a página
- Interface simples e direta

---

## 🔊 Diferencial

- Integração com **responsiveVoice** para feedback por voz  
  (o jogo “fala” com o usuário durante a interação)
